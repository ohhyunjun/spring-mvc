HERV

HERV 동아리에서 미세먼지, 빗물, 조도 센서로 주변 환경을 감지해 창문을 자동 제어하는 스마트 시스템을 제작했습니다. 이 시스템은 외부 유해물질 유입을 막거나 날씨 변화, 실내 밝기에 대응하여 쾌적하고 안전한 실내 환경을 자동으로 조성합니다.

아두이노를 기반으로 모터와 톱니바퀴를 이용한 개폐 장치를 구현했으며, 저는 프로젝트에서 미세먼지 센서의 회로 구성과 코딩을 주로 담당했습니다. 또한, 전체 시스템의 최종 코드를 디버깅하며 안정성을 높이는 역할을 맡아 실질적인 문제 해결 능력을 기를 수 있었습니다.

이 경험은 코딩과 회로 설계를 아우르는 통합적인 문제 해결 능력을 길러주었으며, 아이디어를 하드웨어 프로토타입으로 완성하는 전체 사이클을 깊이 이해하는 계기가 되었습니다.
